# Pano视频通话示例程序

拍乐云Pano 提供了一组示例程序，展示如何集成拍乐云音视频 SDK 以便于用户开发自己的视频通话应用。

## 平台支持

示例程序支持 Windows、Android、iOS、Web、Electron 平台。

## 演示功能

- 加入 / 退出频道
- 一对一 / 多方会议模式选择
- 视频开启 / 关闭
- 视频订阅 / 取消订阅
- 摄像头切换
- 静音 / 取消静音
- 扬声器 / 耳机切换
